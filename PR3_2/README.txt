Корткий опис проєкту PR3_2:
------------------------------------------------------------------------------------------------------
Цей проєкт реалізує багатопоточну програму для пошуку файлів у заданій директорії. Програма рахує
кількість файлів, назви яких містять певну літеру або слово, вказане користувачем. Реалізація
виконується за допомогою Fork/Join Framework з використанням підходу Work Stealing, що забезпечує
ефективний розподіл підзадач між потоками.
------------------------------------------------------------------------------------------------------
Основний функціонал:
 - Пошук файлів у директорії та її піддиректоріях.
 - Підтримка введення користувачем шляху до директорії та пошукового запиту (слова чи літери).
 - Використання рекурсивного підходу для розділення завдань.
 - Висока продуктивність завдяки паралельній обробці задач.
------------------------------------------------------------------------------------------------------
Чому було надано перевагу Fork/Join Framework:
 > Ефективність - Fork/Join Framework оптимізує розподіл підзадач завдяки підходу Work Stealing. Це
дозволяє максимально використовувати доступні потоки й уникати простоїв.
 > Простота реалізації - Fork/Join Framework надає інтуїтивний API для рекурсивного розділення завдань,
що зменшує складність коду.
 > Продуктивність - Завдяки паралельній обробці підзадач, програма виконується швидше, особливо на
великих наборах файлів у глибоких директоріях.
